## 🌟 پروکسی پرو ادونس چیست؟

یک ابزار خط فرمان (CLI) حرفه‌ای که به شما کمک می‌کند:

- 🔎 **جمع‌آوری** هزاران پروکسی از منابع عمومی و خصوصی
- ⚙️ **تست سریع** با کانکارنسی هوشمند
- 🎯 **فیلتر و مرتب‌سازی** پروکسی‌های واقعاً سالم
- 🔐 **پشتیبانی کامل** از پروکسی‌های یوزر و پسورد دار
- 📁 **خروجی تمیز و آماده استفاده** در فایل‌های جداگانه

چه یک **کاربر مبتدی** باشید که اولین بار تست می‌کنید،
چه یک **کاربر حرفه‌ای** که به کانکارنسی بالا و فیلتر جغرافیایی نیاز دارید —
این ابزار برای شما ساخته شده! 💪

## ✨ ویژگی‌های کلیدی

| 🚀 قابلیت | 📖 توضیح |
|---|---|
| **کالکتور هوشمند** | جمع‌آوری از منابع عمومی به‌روز |
| **منابع خصوصی** | افزودن لیست‌های شخصی شما (با یا بدون یوزر/پسورد) |
| **تست دو مرحله‌ای** | ابتدا TCP سریع، سپس بررسی کامل HTTP |
| **تشخیص آنانیمیتی** | Elite / Anonymous / Transparent |
| **تشخیص کشور** | نمایش کشور هر پروکسی سالم |
| **پروکسی‌های احراز هویت‌دار** | مدیریت کامل `user:pass@host:port` |
| **فایل مجزا برای Auth** | خروجی جداگانه برای پروکسی‌های یوزر/پسورد دار |
| **حالت‌های Balanced / Fast / Deep** | انتخاب بین سرعت و دقت |
| **رابط زیبا** | منوی رنگی مدرن با کلیدهای میانبر |

## 📦 نصب (مخصوص افراد مبتدی)

### 1️⃣ نصب Node.js (فقط یک بار)
نسخه **LTS** را از [nodejs.org](https://nodejs.org) دانلود و نصب کنید.

### 2️⃣ دانلود این ابزار
از گیت‌هاب دکمه سبز **Code → Download ZIP** را بزنید و فایل را از حالت زیپ خارج کنید.

### 3️⃣ اجرای نصب‌کننده (ویندوز)
روی **`INSTALL_FIRST.bat`** دو بار کلیک کنید و چند ثانیه صبر کنید.

### 4️⃣ اجرای ابزار
روی **`RUN.bat`** دو بار کلیک کنید — تمام! 🎉

> 💡 **کاربران لینوکس / مک:** ترمینال را در پوشه باز کنید و اجرا کنید:
> ```bash
> npm install
> node proxy-pro.js
> ```

## 🎮 معرفی منوی اصلی

```
┌────────────────────────────────────────────────┐
│   Mode : balanced  Two-Phase: on  Conc: 150    │
│   Anon: on  Geo: on                            │
├────────────────────────────────────────────────┤
│  1) جمع‌آوری پروکسی (هوشمند / خصوصی / هر دو)   │
│  2) تست پروکسی‌ها                              │
│  3) مشاهده نتایج                               │
│  4) تنظیمات                                    │
│  H) راهنمای کامل                               │
│  0) خروج                                       │
└────────────────────────────────────────────────┘
```

### 🧭 استفاده سریع
1. کلید **`1`** را برای جمع‌آوری بزنید (هوشمند / خصوصی / هر دو)
2. کلید **`2`** را برای تست بزنید
3. کلید **`3`** را برای مشاهده پوشه نتایج بزنید

## 📂 فایل‌های خروجی

بعد از پایان تست، در پوشه `results/` این فایل‌ها را می‌بینید:

| 📄 فایل | 🎯 محتوا |
|---|---|
| `alive.txt` | تمام پروکسی‌های **عمومی سالم** (بدون احراز هویت) |
| `alive_with_auth.txt` | پروکسی‌های سالم **یوزر/پسورد دار** با نمایش **کامل** اطلاعات |
| `dead.txt` | پروکسی‌های خراب |
| `summary.txt` | آمار کامل تست |
| `by_country/` | تفکیک پروکسی‌ها بر اساس کشور |
| `by_anonymity/` | تفکیک بر اساس سطح آنانیمیتی |

> 🔒 **نکته حریم خصوصی:** فایل `alive_with_auth.txt` اطلاعات کامل یوزر و پسورد را نمایش می‌دهد چون این فایل مخصوص خود شماست. آن را در جای عمومی به اشتراک نگذارید!

## ⚙️ تنظیمات

با ویرایش فایل **`config.json`** می‌توانید تنظیمات را تغییر دهید:

- 🧵 `concurrency` — تعداد تست موازی (پیش‌فرض `150`)
- ⏱️ `timeoutMs` — زمان تایم‌اوت هر درخواست
- 🎚️ `mode` — حالت `fast` / `balanced` / `deep`
- 🌍 `geo` — فعال/غیرفعال کردن تشخیص کشور
- 🕶️ `anonymity` — فعال/غیرفعال کردن تشخیص آنانیمیتی
